# Logo do projeto

Este diretório está preparado para receber a logo final em SVG.

## Como usar
1. Baixe a logo escolhida (segunda opção do Zen).
2. Converta para SVG (se necessário).
3. Substitua este arquivo: `public/brand/logo.svg`
4. Mantenha exatamente esse nome (`logo.svg`).

Pronto: o header do site já usa automaticamente `/brand/logo.svg`.
